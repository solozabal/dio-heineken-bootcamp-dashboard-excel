# Dashboard de Assinaturas

Este projeto é um dashboard interativo desenvolvido com Streamlit para visualizar e analisar dados de assinaturas.

## Estrutura do Projeto

- `.streamlit/config.toml`: Configurações de tema para o Streamlit.
- `app.py`: Código principal do dashboard.
- `database.xlsx`: Base de dados utilizada pelo dashboard.
- `requirements.txt`: Lista de dependências do projeto.

## Observação Importante

O arquivo `.dashboard-finalizado` é um arquivo .xlsx feito somente para entrega da atividade do Bootcamp.

## Como Executar

1. Clone o repositório:
    ```sh
    git clone https://github.com/seu-usuario/seu-repositorio.git
    cd seu-repositorio
    ```

2. Crie e ative um ambiente virtual:
    ```sh
    python -m venv venv
    source venv/bin/activate  # No Windows use `venv\Scripts\activate`
    ```

3. Instale as dependências:
    ```sh
    pip install -r requirements.txt
    ```

4. Execute o aplicativo:
    ```sh
    streamlit run app.py
    ```
